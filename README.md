# ForBoot-Components
## 说明

## 快速上手
安装集成要求如下：
- JDK 17
- Maven or Gradle

## ForBoot Components
-  [ForBoot-OSS](https://github.com/focre-projects/forboot-components/tree/main/forboot-oss) 文件存储,支持选择使用不同存储方式，也可以同时使用多种存储方式

| 类型          | 说明  |
|-------------|-----|
| local       | 本地存储 |
| minio       | MINIO分布式存储 |
| aliyun      | 阿里云OSS存储 |
| tencent_cos | 腾讯云COS存储 |
| aws_s3      | AWS S3存储 |
| qiniu_kodo  | 七牛云存储 |
